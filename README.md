# Omakase Today – Smart Recipe Recommendation System

## 📝 Project Description
Omakase Today is a web-based system designed to solve the "what should I eat?" dilemma. It provides users with smart recipe suggestions based on available ingredients, search queries, or complete randomness.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS
* **Database:** SQLite

## ✨ Key Features
* **Smart Recommendations:** Get recipes based on ingredients you already have.
* **Randomizer:** Let the system choose a meal for you.
* **Recipe Management:** Admin tools to add, edit, and delete recipes.
* **User Accounts:** Personalized experience for saving favorite meals.
