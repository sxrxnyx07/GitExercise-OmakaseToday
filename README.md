# 🥗 Ingredient Matching & Search System (Member 3)
**Developer:** SARANYA A/P RAVI

This branch covers the development of the following core features for the **Omakase Today** project:

## 🛠️ Individual Task Scope

### 1. Ingredient-Based Recommendation
* **Input Handling:** Allow users to input ingredients they already have.
* **Matching Logic:** Recommend recipes that match the available ingredients.
* **Data Display:** Show missing ingredients and calculate the matching percentage.

### 2. Recipe Search System
* **Keyword Search:** Implement a search bar for users to find specific recipes.
* **Search Filtering:** Filter and display recipes based on the search results.

### 3. Flavor Preference Filter
* **Flavor Categories:** Support for **Light, Spicy, Sour, and Sweet** types.
* **Preference Logic:** Filter recipes based on the user's selected flavor preference.
